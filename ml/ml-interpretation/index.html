Statistical Model Interpretation

May 2, 2022 · 4 min read
Minjun Park

Machine learning(ML) is an approach to learn some pattern of data, and leverage it to infer unseen data. The same statement is valid for statistics. This is because the essence of ML theory comes from statistics. In this post, I will explain how to interpret the output of ML models from a statistical perspective.

Deterministic ML Interpretation​

Use the output of the model as a solution to the problem. Since the model is a deterministic function, it is a point estimate without stochasticity at inference time.

ComponentInterpretation
Modeldeterministic function fθ(⋅)f_\theta(\cdot)fθ​(⋅) which returns a deterministic output given a input xxx
Lossdifference between the answer yyy and the output of the model fθ(x)f_\theta(x)fθ​(x)
Objectivefind a model, which captures patterns in the given dataset D=(x1,y1),...,(xN,yN)D={(x_1, y_1),...,(x_N, y_N)}D=(x1​,y1​),...,(xN​,yN​)
Optimizationtrain to make the output, fθ(x)f_\theta(x)fθ​(x), same as the label, yyy.

Assumptions​

The goal is to answer "What would be the outcome Y given the input X?". This is equivalent to estimating the conditional probability distribution P(Y∣X)P(Y\vert X)P(Y∣X). Since estimating the true distribution P(Y∣X)P(Y\vert X)P(Y∣X) is intractable, we approximate it with a parametrized distribution P(Y∣paramter w)P(Y\vert \text{paramter } w)P(Y∣paramter w). The functional form of the parametrized distribution is our design decision.

In statistical interpretation, we treat the output of the ML model,fθ(X)f_\theta (X)fθ​(X), as the parameter www in the approximate distribution. The resulting distribution P(Y∣w=fθ(X))P(Y\vert w=f_\theta (X))P(Y∣w=fθ​(X)) is considered as the conditional probability distribution P(Y∣X)P(Y\vert X)P(Y∣X).

note
  • The parameter θ\thetaθ is the parameter for the ML model e.g. weights in nueral networks.
  • The parameter www is the parameter for the conditional probability that we defined.

Define Loss Function​

Let's consider an i-th sample (xi,yi)(x_i, y_i)(xi​,yi​). Given the input xix_ixi​, the probability distribution on yyy is P(Y∣fθ(xi))P(Y\vert f_\theta(x_i))P(Y∣fθ​(xi​)). The probability to observe the sample iii is P(Y=yi∣fθ(xi))P(Y=y_i\vert f_\theta(x_i))P(Y=yi​∣fθ​(xi​)). This is the probability to observe yiy_iyi​ given a target variable θ\thetaθ, which is likelihood. To ease the notation, we denote P(Y=yi∣fθ(xi))=p(yi∣fθ(xi))P(Y=y_i\vert f_\theta(x_i))=p(y_i\vert f_\theta(x_i))P(Y=yi​∣fθ​(xi​))=p(yi​∣fθ​(xi​)) for the remaining post. Since the samples are in i.i.d. condition and we know the likelihood for a sample, we can calculate the likelihood for the whole data.

P(D∣θ)=∏iP(Y=yi∣X=xi,θ)=∏ip(yi∣fθ(xi))P(D|\theta)=\prod_i P(Y=y_i|X=x_i,\theta)=\prod_i p(y_i|f_\theta (x_i))P(D∣θ)=i∏​P(Y=yi​∣X=xi​,θ)=i∏​p(yi​∣fθ​(xi​))

The goal is to maximize the likelihood. Considering loss functions must be represented as a sum of terms, and it is preferred to minimize the loss, we use negative log likelihood(NLL) as the loss function.

max⁡θ∏ip(yi∣fθ(xi))  ⟺  max⁡θ∑ilog⁡p(yi∣fθ(xi))  ⟺  min⁡θ∑i−log⁡p(yi∣fθ(xi))\begin{align*} \max_\theta \prod_i p(y_i|f_\theta(x_i)) &\iff \max_\theta \sum_i \log p(y_i|f_\theta(x_i)) \\ &\iff \min_\theta \sum_i -\log p(y_i|f_\theta(x_i)) \end{align*}θmax​i∏​p(yi​∣fθ​(xi​))​⟺θmax​i∑​logp(yi​∣fθ​(xi​))⟺θmin​i∑​−logp(yi​∣fθ​(xi​))​

Now, let's go back to the parametrized distribution we defined. P(Y∣fθ(X))P(Y|f_\theta(X))P(Y∣fθ​(X)) means that the label yyy is non-deterministic although the ML model fθ(x)f_\theta(x)fθ​(x) is deterministic. If we assume that the label has a Gaussian noise around a deterministic function, we can rewrite the distribution as below:

P(Y∣fθ(X))=N(Y∣μ(X),σ)P(Y\vert f_\theta(X)) = \mathcal{N}\Big(Y\vert \mu(X), \sigma\Big)P(Y∣fθ​(X))=N(Y∣μ(X),σ)

Since the variance is fixed, the only parameter required is μ\muμ. By replacing the μ(X)\mu(X)μ(X) with the ML model output fθ(X)f_\theta(X)fθ​(X), we can derive that minimizing the negative log likelihood is equivalent to minimizing the squared error. reference

−log⁡p(yi∣fθ(xi))=−log⁡[N(yi∣fθ(xi),σ)]∝(yi−fθ(xi))2∴loss∝∑i(yi−fθ(xi))2∝1N∑i(yi−fθ(xi))2=mean squared error\begin{align*} -\log p(y_i|f_\theta(x_i)) &= -\log [\mathcal{N}\Big(y_i\vert f_\theta(x_i), \sigma\Big)] \\ &\propto \big(y_i-f_\theta(x_i)\big)^2\\ \therefore loss &\propto \sum_i \big(y_i-f_\theta(x_i)\big)^2\\ &\propto \frac{1}{N}\sum_i \big(y_i-f_\theta(x_i)\big)^2 = mean\ squared\ error \end{align*}−logp(yi​∣fθ​(xi​))∴loss​=−log[N(yi​∣fθ​(xi​),σ)]∝(yi​−fθ​(xi​))2∝i∑​(yi​−fθ​(xi​))2∝N1​i∑​(yi​−fθ​(xi​))2=mean squared error​
