[Optimization] MLE & MAP

February 9, 2023 · 3 min read
Minjun Park

One of the most important tasks in ML is to optimize parameters in a model. There are various approaches and I'll explain the most fundamental methods (MLE & MAP) in this post.

context​

These concepts works on a [[Probabilistic Model]]. We assume that the data distribution is parametrized by θ\thetaθ and the likelihood is defined as below:

P(x∣θ)=∏ip(xi∣θ)P(\mathbf{x}\vert\theta) = \prod_i p(x_i\vert \theta)P(x∣θ)=i∏​p(xi​∣θ)

Optimization Methods​

MLE​

Maximum Likelihood Estimation(MLE) maximizes the likelihood P(D∣θ)P(D\vert\theta)P(D∣θ) by optimizing the model parameter θ\thetaθ. The result of MLE is a point estimation of the model parameter.

θMLE=arg⁡max⁡θP(x∣θ)=arg⁡max⁡θ∏ip(xi∣θ)\theta_{MLE}=\arg\max_\theta P(\mathbf{x}\vert\theta)=\arg\max_\theta\prod_i p(x_i\vert\theta)θMLE​=argθmax​P(x∣θ)=argθmax​i∏​p(xi​∣θ)

This fits the model to the observed data. However, MLE is susceptible to overfitting the model when the data is insufficient. For example, some rare events might not be included in the dataset. MLE assigns zero probability to the unobserved events.

MAP​

Unlike MLE, MAP utilizes the prior knowledge on the data distribution. The prior knowledge is represented as a distribution on the parameter θ\thetaθ, P(θ)P(\theta)P(θ). Given the observations DDD, we can update the knowledge on θ\thetaθ by applying Bayes' theorem.

P(θ∣x)=P(x∣θ)P(θ)P(x)∝P(x∣θ)P(θ)P(\theta\vert \mathbf{x})=\frac{P(\mathbf{x}\vert\theta)P(\theta)}{P(\mathbf{x})} \propto P(\mathbf{x}\vert\theta)P(\theta)P(θ∣x)=P(x)P(x∣θ)P(θ)​∝P(x∣θ)P(θ)

With the updated distribution on θ\thetaθ, the most plausible value is arg⁡max⁡θP(θ∣x)\arg\max_\theta P(\theta\vert \mathbf{x})argmaxθ​P(θ∣x). This formula can be rewritten in terms of likelihood:

θMAP=arg⁡max⁡θP(θ∣x)=arg⁡max⁡θ∏ip(xi∣θ)p(θ)\theta_{MAP}=\arg\max_\theta P(\theta\vert \mathbf{x})=\arg\max_\theta \prod_i p(x_i\vert\theta)p(\theta)θMAP​=argθmax​P(θ∣x)=argθmax​i∏​p(xi​∣θ)p(θ)
Relationship between MLE and MAP

If the prior distribution on θ\thetaθ is an uniform distribution, MAP converges to MLE.

MLE with Latent Variable model​

In a simple model without latent variables, MLE finds the optimal θ\thetaθ that maximizes the likelihood pθ(x)p_\theta(\mathbf{x})pθ​(x). Now, we introduced latent variables z\mathbf{z}z to the model. We can still apply MLE by maximizing the likelihood if it is tractable.

log likelihood=log⁡pθ(x)=log⁡∫pθ(x,z)dz\begin{align} \text{log likelihood} &= \log p_\theta (\mathbf{x})\\ &= \log \int p_\theta(\mathbf{x},\mathbf{z}) d\mathbf{z} \end{align}log likelihood​=logpθ​(x)=log∫pθ​(x,z)dz​​

However, MLE is inapplicable in most cases. There are two main reasons why we can't use MLE:

  1. Even for a fixed θ\thetaθ, the integral over latent variable zzz is often intractable. (Evaluating likelihood is intractable.)
  2. Even if we can calculate the log likelihood for all values of θ\thetaθ, maximizing the log likelihood as a function of θ\thetaθ can be a difficult non-convex optimization problem. (Cannot guarantee to find the global optimum.)

Even if we find an optimal model parameter θmax\theta_{max}θmax​ somehow, we cannot use it for prediction. To construct the posterior predictive distribution pθmax(xnew∣x)p_{\theta_{max}}(x_{new}\vert \mathbf{x})pθmax​​(xnew​∣x), we need the posterior on the latent variable pθmax(z∣x)p_{\theta_{max}}(\mathbf{z} \vert \mathbf{x})pθmax​​(z∣x). However, the denominator of the posterior is the likelihood pθmax(x)=∫pθmax(x,z)dzp_{\theta_{max}}(\mathbf{x})=\int p_{\theta_{max}}(\mathbf{x},\mathbf{z}) d\mathbf{z}pθmax​​(x)=∫pθmax​​(x,z)dz, which is intractable in general.
